This is a personal knowledge vault ("second brain") managed in [Obsidian](https://obsidian.md). AI agents read from and write to it continuously — see `CLAUDE.md` for agent conventions and rules.

## Folder structure

| Folder | Purpose |
| --- | --- |
| `01 Updates/` | Daily research updates by topic — one file per topic, newest entries at top |
| `02 Daily/YYYY/MM/` | Daily notes, named `MM-DD-YY ddd.md` |
| `03 Meetings/` | Call and meeting notes, organized by year; transcripts in `Transcriptions/` |
| `04 People/` | One note per person; people with attached docs get a folder |
| `05 Projects/` | One folder per project with a root note and standard frontmatter |
| `06 Research/` | Deep research documents |
| `07 References/` | Clippings, articles, external knowledge, tool docs |
| `08 Summaries/` | Notes generated by the `summarize` and `summarize-call` skills |
| `10 Places/` | Places — cafes, restaurants, gyms, hotels, homes |
| `Archive/` | Completed or inactive projects |
| `_Attachments/` | All media files — audio, images, PDFs, ebooks |
| `_Bases/` | Obsidian Bases (`.base` files) |
| `_Templates/` | Reusable note templates |

## Conventions

- No `# Title` headings — the filename is the title
- All notes carry `unread: true` frontmatter when created or edited; the `unread-dot` plugin clears it on open
- Relationships use backlinks, not tags
- Agent rules live in `CLAUDE.md`
