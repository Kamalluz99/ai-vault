## iMessage management

- Created [[iMessage]] project in `05 Projects/iMessage/`
- Built `imessage.sh` — macOS CLI with 6 subcommands: `send`, `read`, `list`, `search`, `unread`, `export`
- Sending uses AppleScript via `osascript`; reading queries `~/Library/Messages/chat.db` via `sqlite3`
- `export` command pulls conversations from `chat.db` and writes Obsidian-formatted notes to `03 Meetings/iMessage/`; auto-links contacts to `04 People/` notes when a match exists
