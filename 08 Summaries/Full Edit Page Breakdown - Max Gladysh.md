## Summary

A comprehensive 12-block breakdown of the [[DaVinci Resolve]] Edit page, focused on efficient, Fusion-free editing workflows. [[Max Gladysh]] walks through everything from essential shortcuts to final color and sound polish, with an emphasis on timeline organisation, B-roll integration, and motion design using only the Edit page.

---

## Key Takeaways

- **Shortcuts first** — learning the Edit page shortcuts is the foundation; covered before anything else
- **Beginner overview is deep** — the beginner block runs nearly 18 minutes and covers the full Edit page interface
- **Cleaning footage** — dedicated block on trimming, organising, and preparing raw footage before editing
- **Elements & Backgrounds** — how to layer visual assets (graphics, overlays, backgrounds) without Fusion
- **Effects** — practical effects workflow entirely within the Edit page (~15 min block)
- **Animation** — motion and animation techniques (~12 min block), again without Fusion
- **Text, Transitions, Texture** — separate blocks for each, showing how to build polished visuals systematically
- **Sound & Color last** — polish comes at the end once the edit is locked
- **Style block** — the final creative layer, tying the visual identity together

## DaVinci Resolve Edit Page Shortcuts

### Basics
| Shortcut | Action |
|----------|--------|
| `Cmd+Z` | Undo |
| `Cmd+Shift+Z` | Redo |
| `Cmd+X` | Cut |
| `Cmd+C` | Copy |
| `Cmd+V` | Paste |
| `Opt+V` | Paste Attributes |
| `Cmd+A` | Select All |
| `Cmd+Shift+A` | Deselect All |

### Tools
| Shortcut | Action |
|----------|--------|
| `A` | Pointer (Selection) tool |
| `T` | Trim mode |
| `B` | Blade tool |
| `Cmd+B` | Razor (split at playhead) |
| `W` | Dynamic trim mode |

### Editing
| Shortcut | Action |
|----------|--------|
| `F9` | Insert edit |
| `F10` | Overwrite edit |
| `F11` | Replace edit |
| `F12` | Place on top edit |
| `Backspace` | Delete |
| `Cmd++` | Split clip |
| `Opt++` | Join clip |
| `Cmd+T` | Add transition |
| `Y` | Select clips forward |
| `U` | Toggle edit point type |
| `N` | Snapping on/off |

### Playback & Navigation
| Shortcut | Action |
|----------|--------|
| `L` | Play forward |
| `J` | Play reverse |
| `K` | Stop |
| `;` | Go to first frame |
| `'` | Go to last frame |
| `[` | Go to previous keyframe |
| `]` | Go to next keyframe |
| `↓` | Next clip |
| `↑` | Previous clip |

### Marking
| Shortcut | Action |
|----------|--------|
| `I` | Set mark in |
| `O` | Set mark out |
| `X` | Mark clip |
| `M` | Add marker |

### File
| Shortcut | Action |
|----------|--------|
| `Cmd+S` | Save |
| `Cmd+Shift+S` | Save As |
| `Cmd+N` | New Timeline |
| `Cmd+I` | Import |
| `Cmd+E` | Export |

> **Tip:** Customise shortcuts via DaVinci Resolve → Keyboard Customization. You can import Premiere Pro or Avid layouts directly.
