A second brain is a personal knowledge management system that externalises memory and thinking into a structured, searchable digital store. The concept was popularised by [[Tiago Forte]] through his [[Building a Second Brain]] book and course, and is associated with methods like [[PARA]] (Projects, Areas, Resources, Archives). In the context of AI-managed vaults, the second brain is extended by [[LLM]] agents that autonomously populate it — summarising books, logging meetings, updating project notes — so the system grows without manual capture overhead. Tools like [[Obsidian]] are the most common substrate; [[Claude Code]] is increasingly the agent layer that manages them.
