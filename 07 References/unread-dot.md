unread-dot is a custom [[Obsidian]] plugin made by [[Reysu]] that displays a blue dot in the file explorer next to any note with `unread: true` in its frontmatter. When an AI agent creates or modifies a note, it sets `unread: true`; the dot appears immediately. Opening the note clears the dot (sets `unread: false`). This gives a lightweight at-a-glance signal of what the agent changed without manually checking every file. Part of the [[ai-life-skills]] toolkit; available in the repository's `obsidian-plugins/` folder.
