Claude Code is [[Anthropic]]'s official agentic CLI tool that brings [[Claude]] into the terminal as an interactive coding and task-execution agent. It can read, write, and edit files; run shell commands; search the web; call MCP tools; and spawn sub-agents for parallel workloads. In the context of an [[Obsidian]] vault, Claude Code acts as the AI brain — maintaining notes, summarizing content, updating daily logs, running scheduled routines (via `/schedule`), and following vault-level instructions defined in [[CLAUDE.md]]. It supports both cloud (Anthropic API) and local model configurations.
