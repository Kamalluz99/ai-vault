LM Studio is a free desktop application (Mac, Windows, Linux) for downloading, managing, and running [[open source models]] locally. It provides a [[ChatGPT]]-style chat UI and an OpenAI-compatible local API server, making local models drop-in replacements for cloud APIs in most apps and scripts. Models are sourced from [[Hugging Face]] and run entirely on your hardware — no internet required after download. It supports quantised models (GGUF format via llama.cpp) optimised for Apple Silicon, NVIDIA GPUs, and CPU-only machines. Recommended by [[Reysu]] as the primary tool for privacy-preserving AI tasks in a local-first [[second brain]] setup.
