Open source models are [[LLM]]s whose weights are publicly released, allowing anyone to download and run them locally. Leading families include Meta's Llama series, Mistral, Qwen (Alibaba), Gemma (Google), and Phi (Microsoft). They are typically run via tools like [[LM Studio]] (desktop UI) or Ollama (CLI). Quantised versions (GGUF format) run on consumer hardware including Apple Silicon. Key advantages over [[frontier models]]: zero data-sharing with third parties, no ongoing API costs, and full offline operation. Limitations: generally lower reasoning capability than frontier models, though the gap narrows for constrained tasks. Essential for privacy-sensitive workloads in an AI-managed [[second brain]].
